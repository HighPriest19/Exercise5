# Exercise 5 - C# Console Application

**A simple interactive console program** built as part of the CSE124 / BYU-Pathway C# course.

![C# Logo](https://upload.wikimedia.org/wikipedia/commons/thumb/b/bd/Logo_C_sharp.svg/256px-Logo_C_sharp.svg.png)

---

## 📋 Overview

This program demonstrates basic C# console input/output, string interpolation, and simple arithmetic.

**What it does:**
- Greets the user
- Asks for their name
- Asks for their favorite number
- Calculates the square of the number
- Displays a personalized message

---

## 🛠️ Features

- User input handling (`Console.ReadLine()`)
- Type conversion (`int.Parse()`)
- String interpolation (`$"..."`)
- Basic math operations

---

## 🚀 How to Run

### Prerequisites
- [.NET 9 SDK](https://dotnet.microsoft.com/download) (or later)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/HighPriest19/Exercise5.git
   cd Exercise5
   ```

2. **Build and run**
   ```bash
   dotnet run
   ```

   Or open the solution in Visual Studio / VS Code and run it.
